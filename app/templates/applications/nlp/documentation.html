Natural Language Processing

Multilingual Sentiment Analysis Using Machine Learning and Generative AI

This project demonstrates how Natural Language Processing (NLP), Machine Learning, and Generative AI can analyze human language, understand emotions, and provide intelligent explanations.

The system supports multilingual text input by automatically translating non-English text before sentiment classification.

Sentiment Categories:
  • Positive Sentiment
  • Neutral Sentiment
  • Negative Sentiment
Integrated Technologies:
  • Language Translation
  • TF-IDF Feature Extraction
  • Logistic Regression Classification
  • Gemini AI Interpretation

Knowledge Discovery in Databases (KDD) Process

The system followed the KDD methodology to transform raw text information into meaningful sentiment knowledge.

  1. Data Selection: Twitter sentiment dataset collection.
  2. Preprocessing: Cleaning, normalization, and preparation of text data.
  3. Transformation: TF-IDF converts text into numerical machine learning features.
  4. Data Mining: Classification using different machine learning algorithms.
  5. Evaluation: Performance comparison using evaluation metrics.

Machine Learning Model Evaluation Results

Multiple algorithms were evaluated using Orange Data Mining. Logistic Regression achieved the best overall performance.

Selected Model: Logistic Regression

The trained Logistic Regression model was exported as sentiment_model.pkl and integrated into the Flask AI application.

Hybrid NLP Sentiment Analysis Architecture

The architecture combines language translation, machine learning sentiment classification, and generative AI interpretation.

System Pipeline:
  • User Text Input
  • Automatic Language Detection
  • Tagalog / Multilingual Translation into English
  • TF-IDF Vectorization using tfidf_vectorizer.pkl
  • Sentiment Classification using sentiment_model.pkl
  • Gemini AI Explanation Generation
  • Local Researcher AI Fallback Explanation

Transforming Human Language into Intelligent Insights

This NLP application demonstrates how Artificial Intelligence can understand multilingual text using Machine Learning and Generative AI.

By combining translation processing, TF-IDF feature extraction, Logistic Regression classification, Gemini AI interpretation, and local AI fallback, the system provides meaningful human-readable sentiment analysis.
