📊 Buy Computer Prediction

A Predictive Analytics application that evaluates input characteristics and predicts possible outcomes using a trained machine learning model.

Random Forest • Flask • Gemini AI

💬 Twitter Sentiment Analysis

A Natural Language Processing system that analyzes text input and determines whether the expressed sentiment is positive, neutral, or negative.

Logistic Regression • TF-IDF • Gemini AI

A Computer Vision application that analyzes uploaded images and predicts whether the image belongs to a cat or dog category.

MobileNetV2 CNN • TensorFlow • Gemini AI
