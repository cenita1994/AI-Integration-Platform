Natural Language Processing

Twitter Sentiment Analysis Using Machine Learning

This project demonstrates how Natural Language Processing (NLP) and Machine Learning can analyze text data and identify user emotions, opinions, and feedback.

The system classifies Twitter text into three sentiment categories:

  • Positive Sentiment
  • Neutral Sentiment
  • Negative Sentiment

Google Drive Files Google Colab Notebook Orange Data Mining

Dataset Description

The dataset used in this project contains Twitter text data labeled according to user sentiment. Each record contains a text statement and its corresponding sentiment classification.

Sentiment Number of Records
Neutral 11,118
Positive 8,582
Negative 7,781
Total 27,481 Tweets

The dataset allows the machine learning model to learn different language patterns, emotions, and expressions commonly found in online text communication.

Knowledge Discovery in Databases (KDD) Process

The development followed the KDD methodology to transform raw text data into meaningful sentiment knowledge.

  1. Data Selection: Twitter text dataset was collected.
  2. Data Preprocessing: Text cleaning and preparation were performed.
  3. Data Transformation: Text data was converted into numerical features using TF-IDF.
  4. Data Mining: Machine Learning algorithms were applied for sentiment classification.
  5. Evaluation: Model performance was compared using different evaluation metrics.

Orange Data Mining Workflow

Orange Data Mining was used to visually design and evaluate the Natural Language Processing workflow.

The text data was processed using feature extraction techniques and evaluated using multiple machine learning algorithms.

  • Naive Bayes
  • Support Vector Machine
  • Random Forest
  • Logistic Regression

Machine Learning Model Evaluation Results

Several machine learning algorithms were evaluated to determine the most suitable model for Twitter sentiment classification. The models were compared using AUC, Classification Accuracy, F1 Score, Precision, Recall, and MCC.

Model AUC CA F1 Precision Recall MCC
Naive Bayes 0.6999 0.5150 0.5152 0.5248 0.5150 0.2759
SVM 0.6774 0.4260 0.3837 0.4824 0.4260 0.1858
Random Forest 0.7152 0.5515 0.5386 0.5837 0.5515 0.3216
Logistic Regression 0.7616 0.5865 0.5768 0.6255 0.5865 0.3790

Selected Model: Logistic Regression

Logistic Regression achieved the strongest overall performance among the evaluated algorithms. It obtained the highest AUC, Classification Accuracy, Precision, F1 Score, and MCC values.

The result indicates that Logistic Regression provided the best balance between correct classification and reliability in identifying positive, neutral, and negative text sentiments.

Google Colab Model Development

Google Colab was used to develop and export the final machine learning model used by the web application.

The text data was converted into numerical features using TF-IDF Vectorization before training the Logistic Regression classification model.

Generated Files:
  • tfidf_vectorizer.pkl - converts text into numerical features
  • sentiment_model.pkl - predicts sentiment classification

Sentiment Analysis System Architecture

The implemented system integrates Natural Language Processing, Machine Learning, and Generative AI to analyze user text input.

The user enters text in any supported language. The system translates the input into English before processing.

TF-IDF converts the text into numerical features that can be understood by the trained Logistic Regression model.

The system predicts whether the message expresses a positive, neutral, or negative sentiment. Generative AI is then used to provide a human-readable explanation of the result.

Transforming Text into Meaningful Insights

This Natural Language Processing application demonstrates how machine learning can analyze human language and automatically understand opinions, emotions, and feedback from text data.

By combining language translation, TF-IDF feature extraction, Logistic Regression classification, and AI interpretation, the system provides intelligent sentiment analysis that supports better understanding of user-generated content.
