Cat vs Dog Image Classification

An Artificial Intelligence application that utilizes Computer Vision and Deep Learning techniques to classify images of cats and dogs using a Convolutional Neural Network model.

The system combines image preprocessing, transfer learning using MobileNetV2 CNN architecture, Flask web integration, and AI-based interpretation to provide understandable prediction results.

Computer Vision Deep Learning CNN Model

Project Overview

This project demonstrates the implementation of an AI-powered image classification system capable of identifying whether an uploaded image belongs to a cat or dog category.

The development process follows a machine learning workflow, starting from dataset preparation, model training, evaluation, deployment, and integration into a web-based prediction system.

Technologies Used:
  • Python Programming Language
  • TensorFlow and Keras Deep Learning Framework
  • MobileNetV2 Transfer Learning Model
  • Orange Data Mining for Visual AI Workflow
  • Flask Web Framework
  • Gemini AI for Prediction Explanation

Dataset Preparation

The dataset contains labeled images of cats and dogs that were organized into separate classes for supervised image classification.

Orange Data Mining Workflow

Orange Data Mining was used as a visual machine learning environment to demonstrate the process of image loading, feature extraction, model evaluation, and prediction.

The workflow provides an interactive representation of the image classification pipeline without manually coding every machine learning process.

Model Training Using Google Colab

The deep learning model was developed and trained using Google Colab with TensorFlow and Keras libraries.

Google Colab provides cloud-based GPU resources that allow efficient training of convolutional neural networks.

MobileNetV2 CNN Model Architecture

The classification model uses MobileNetV2, a lightweight Convolutional Neural Network architecture designed for efficient image recognition tasks.

Through transfer learning, the model utilizes previously learned visual features and adapts them for the classification of cat and dog images.

Model Configuration
  • Base Model: MobileNetV2
  • Learning Approach: Transfer Learning
  • Classification Type: Binary Classification
  • Output Classes: Cat and Dog

System Architecture and Prediction Flow

The system follows a complete computer vision pipeline where the uploaded image is processed before being analyzed by the trained deep learning model.

Prediction Workflow

  • 2. Image Preprocessing

  • 4. Cat or Dog Prediction
  • 5. Confidence Score Generation
  • 6. Gemini AI Interpretation

AI-Based Interpretation

Aside from displaying the prediction result, the system integrates Gemini Artificial Intelligence to generate a human-readable interpretation of the classification output.

This feature improves user understanding by explaining why the model identified the uploaded image as a cat or dog, making the prediction result more informative and interactive.

Conclusion

The Cat vs Dog Image Classification System demonstrates how computer vision, deep learning, and artificial intelligence can be integrated into a web-based application.

Using MobileNetV2 CNN architecture and transfer learning, the system achieved accurate image classification while providing understandable AI-generated explanations.
