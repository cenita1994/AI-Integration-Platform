Based on the experiment results, Support Vector Machine achieved the highest classification accuracy, while Logistic Regression produced the most balanced performance based on F1 Score and MCC. Random Forest was selected for the implementation stage to demonstrate an ensemble learning approach, model exporting, and integration into a Flask-based prediction system.

Result Interpretation

SVM achieved the highest Classification Accuracy (CA) of 64.29%. This means it produced the highest number of correct predictions among the tested algorithms.

Logistic Regression achieved the highest F1 Score and MCC value, indicating a better balance between correct predictions and classification reliability.

Random Forest combines multiple decision trees to generate predictions. Although the evaluation score was lower in this small dataset, it demonstrates an important real-world AI concept: ensemble learning. The trained Random Forest model was exported as a .pkl file and integrated into the Flask application.
